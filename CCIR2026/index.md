# <i class="fas fa-feather-alt"></i>第三十二届全国信息检索学术会议

在数据爆炸与人工智能崛起的时代浪潮中，信息检索正经历着前所未有的变革。全国信息检索学术会议（CCIR）是由中国中文信息学会（CIPS）举办的权威学术会议，持续聚焦前沿研究，推动了相关技术及互联网产业的发展，更是顶尖学者、行业精英交流思想、激发创新的核心平台。
